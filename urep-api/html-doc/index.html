UAS Report (UREP) Service

This service allows an unmanned aircraft system (UAS) Service Supplier (USS) to submit a UREP to the greater UAS Traffic Management (UTM) ecosystem. This also allows USS to share UREPs amongst eachother, allowing for greater operational awareness within the UTM ecosystem.
More information: https://helloreverb.com

get /ureps
Get UREPs (getUrepsUsingGET)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Query parameters

fields (optional)
Query Parameter — A ',' delimited list of field names from the Urep object to include within the results. All other fields will be excluded. Fields must be valid field names from the Urep object. If not present all fields will be returned.
sort (optional)
Query Parameter — The parameter to sort by. Defaults to the time_measured if not present.
sort_d (optional)
Query Parameter — Direction of the sort, defaults to DESC if not present. Must be either ASC or DESC
limit (optional)
Query Parameter — Max number of Ureps to return default: 10 format: int32
offset (optional)
Query Parameter — Offset number of records to return for pagination. default: 0 format: int32
lat (optional)
Query Parameter — Latitude of point for Geographic search. Must be present if lon or radius are present. If none of these fields are present, no geographic filtering will be performed. format: double
lon (optional)
Query Parameter — Longitude of point for Geographic search. Must be present if lat or radius are present. If none of these fields are present, no geographic filtering will be performed. format: double
radius (optional)
Query Parameter — Radius for Geographic search in nautical miles. Must be present if lat or lon are present. If none of these fields are present, no geographic filtering will be performed. format: double
before (optional)
Query Parameter — Upper bound for a time based query on time_measured. Defaults to the current time if not present. format: date-time
after (optional)
Query Parameter — Lower bound for a time based query on time_measured. Defaults to 24 hours prior to the current time if not present. format: date-time

Return type

array[Urep]
