geographyFragmentUsingGET

A GeographyFragment is a portion (fragment) of the GeoJSON geography data type. Allowable values for this type are LineString, MultiPolygon, Point and Polygon. Since UTM supports most but not all GeoJSON data types, UTM defines a a portion (or fragment) of the GeoJSON specification. For adding or modifing an operation, only LineString and Polygon are supported. (For trajectory-based operations, LineString will be more appropriate.) MultiPolygon is not supported for Operation; rather it is used for Surveillance. For Position submission, the only accepted type is Point.
