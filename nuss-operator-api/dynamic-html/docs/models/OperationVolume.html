OperationVolume

  • actual_time_end : Date
    Time that the operational volume was freed for use by other operations.
  • conformance_geography : Polygon
  • conformance_time_begin : Date
    Assigned by UTM System. Conformance time buffer before the submitted begin time.
  • conformance_time_end : Date
    Assigned by UTM System. Conformance time buffer after the submitted end time.
  • effective_time_begin : Date
    Earliest time the operation will use the operation volume.
  • effective_time_end : Date
    Latest time the operation will done with the operation volume.
  • flight_geography : Polygon
  • gufi : String
  • id : String
  • line_of_sight : Boolean
    Whether this volume is fully seen by the operatoer.
  • max_altitude_conform_wgs84_ft : Double
    The maximum altitude assigned and used by the UTM System to check vertical conformance of an operation. Based on UTM Client-provided max altitude.
  • max_altitude_protected_wgs84_ft : Double
  • max_altitude_wgs84_ft : Double
    The maximum altitude for this operation in this operation volume. In WGS84 reference system using feet as units.
  • min_altitude_conform_wgs84_ft : Double
    The minimum altitude assigned and used by the UTM System to check vertical conformance of an operation. Based on UTM Client-provided min altitude.
  • min_altitude_protected_wgs84_ft : Double
  • min_altitude_wgs84_ft : Double
    The minimum altitude for this operation in this operation volume. In WGS84 reference system using feet as units.
  • near_structure : Boolean
    Is this operation volume within 400' of a structure?
  • ordinal : Integer
    This integer represents the ordering of the operation volume within the set of operation volumes. Need not be consecutive integers.
  • protected_geography : Polygon
  • protected_time_begin : Date
    Assigned by UTM System. Protected time buffer before the submitted begin time.
  • protected_time_end : Date
    Assigned by UTM System. Protected time buffer after the submitted end time.
